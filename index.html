class SobreOProjeto

A Maratona da Programação é um programa intensivo e gratuito desenvolvido para capacitar jovens de 16 a 17 anos com habilidades essenciais de programação e lógica computacional.

Em quatro módulos completos, os participantes terão a oportunidade de aprender desde fundamentos de lógica até desenvolvimento de projetos reais, sob a orientação de profissionais experientes do mercado.

Nosso objetivo é descobrir novos talentos, desenvolver o pensamento lógico, introduzir os conceitos essenciais de programação e preparar os participantes para ingressar no mercado de trabalho de tecnologia.

class Inscricao {
    constructor(dados) {
        this.nome = dados.nome;
        this.email = dados.email;
        this.idade = dados.idade;
        this.motivacao = dados.motivacao;
    }

    async enviar() {
        try {
            await this.validar();
            return "Inscrição realizada!";
        } catch (erro) {
            console.error(erro);
        }
    }
}
